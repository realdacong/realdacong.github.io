window.open()打开页面被浏览器拦截的原因及解决方案

发表于 2017-08-01 | 分类于 JavaScript

最近项目中遇到了在使用window.open()方法在新窗口打开页面时该页面会被浏览器拦截（阻止打开）的问题。
这种情况出现在ajax方法内，也就是说，当浏览器检测到非用户直接（主动）操作产生的新打开窗口时，就会判定该窗口不安全，随即对其进行拦截。
